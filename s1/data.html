Data

The S1-INBOX project was carried out under Japan Argo. The profile data (pressure, temperature, salinity, and non-adjusted oxygen data) is available via the Argo Global Data Assembly Centers (GDACs).As described in the Project Abstract, We compared oxygen data from each float with the oxygen data from the temporally and spatially nearest shipboard fast-response oxygen sensor (RINKO III, JFE Advantech Co., Ltd., http://www.jfe-advantech.co.jp/eng/ocean/rinko/rinko3.html). The sensors were calibrated to Winkler oxygen data. The RINKO III is an optical measurement principle sensor with a 1-s response time; this system provides oxygen data at 1-dbar resolution. As a result, we found that the desolved oxygen values measured by the Optode 3380 were lower than those measured by the RINKO III sensor by up to 30 mol/kg. We therefore adjusted the oxygen data obtained by the floats according to the calibration equations proposed by Uchida et al. (2008). We will plan to release the adjusted oxygen data soon.
